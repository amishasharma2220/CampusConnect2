# CampusConnect

CampusConnect is a full-stack university platform built for Manipal University Jaipur (MUJ). It connects students, clubs, events, marketplace, lost & found, campus food, and more — all in one place.

## Features

- Event discovery and registration
- Club profiles and membership
- Student marketplace (buy/sell textbooks, items)
- Lost & Found board
- Campus food ordering
- Real-time notifications
- Certificate generation
- Leaderboard and achievements
- Admin approval workflows
- Mobile app (React Native)

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React, Vite, TypeScript, Tailwind CSS, shadcn/ui |
| Backend | FastAPI, Python 3.11+ |
| Database | PostgreSQL 15 |
| ORM | SQLAlchemy 2.0, Alembic |
| Auth | JWT (access + refresh tokens) |
| Storage | Cloudinary (images), AWS S3 (files) |
| Payments | Razorpay |
| Mobile | React Native, Expo |
| CI/CD | GitHub Actions |
